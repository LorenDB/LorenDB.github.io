WineForXP

This site is under construction.



Welcome to WineForXP, a project to bring modern functionality to Windows XP! Our objective is porting the Wine project (winehq.org) to Windows XP, so that any modern Windows program can be run on Windows XP.



Why WineForXP? Isn't Windows XP dead?

Windows XP may be dead, yes, but there are still plenty of people who use it. Some use it for nostalgic reasons. Others have old software. And a loyal few vow to never stop using XP for their daily driver.

How can this be done?

It may be possible to actually compile Wine binaries onto Windows, but it would require a vast modification to the source code, if not a total rewrite. Instead, I'm focusing on a slightly more Linux-based approach. I'm planning to base WineForXP off of Cygwin or MinGW-w64. These programs are designed to run Linux programs on Windows. For more information, please click on one of the links.

But how does Windows know to run programs with WineForXP?

Well, the short answer is "Windows doesn't". The long answer: WineForXP will include a program manager of sorts. Starting the program manager will give you a list of all of the installed programs. From there, you can run whatever you want. I'm not sure yet if I will write the program manager myself or if I will cop out and use/modify somebody else's program, but it will probably be the latter.

When will WineForXP be available?

Hopefully by the end of 2019, but definitely by the end of 2020. It largely depends on how much of my spare time I devote to WineForXP.
